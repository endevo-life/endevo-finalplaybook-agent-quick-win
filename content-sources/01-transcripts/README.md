# 01 — Transcripts

Upload working session transcripts where Niki delivered Final Playbook.

- Formats: `.txt`, `.docx`, `.vtt`, `.srt`, Otter exports, PDF — whatever you have.
- One session per file is easiest to trace; batch exports are fine too.
- Name them so order/context is clear, e.g. `2025-11-03_session-clientA_sprint1.txt`.

**PII:** these are git-ignored and stay local. Do not rename real client names
into the committed content library — the extraction pass anonymizes them.

What gets mined from these: how Niki sequences the delivery, the exact language/
scripts she uses, which steps she does first, and where situations branch.
